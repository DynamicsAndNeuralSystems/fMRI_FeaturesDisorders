10-fold cross-validated linear SVM

I have chosen to use 10-fold cross validation via manual implementation, as the sample reweighting options in caret were limited and difficult to interpret.

However, there still is a balanced accuracy peak around 0.5, so we move forward with inverse probability reweighting.

Model-free shuffle null distribution

Generating null distributions from model-free shuffles

This first model-free shuffles method is borrowed from Trentâ€™s implementation in theft. With this method, the input class labels (Schz or Control) are randomly shuffled N times, and for each iteration, the classification accuracy and balanced accuracy are calculated. This yields a null distribution of accuracies and balanced accuracies, circumventing the need for running any classification algorithms across iterations.

Here, Iâ€™ve run 100,000 iterations of the model-free shuffle, generating 100,000 null values for Accuracy and Balanced Accuracy, respectively. Since this method is independent of brain region, the same null distribution can be used to compare with each brain region separately.
